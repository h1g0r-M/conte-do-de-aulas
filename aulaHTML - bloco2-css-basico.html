O que é um chamado?

Um chamado é uma solicitação formal registrada no sistema para reportar um problema, pedir um serviço ou tirar uma dúvida. Cada chamado recebe um número único e é acompanhado até a sua resolução.

Tipos de chamado disponíveis:

  • Suporte técnico
  • Solicitação de equipamento
  • Acesso a sistemas
  • Dúvidas administrativas

Como funciona o processo?

  1. Você abre o chamado preenchendo o formulário
  2. O sistema gera um número de protocolo
  3. Um técnico é designado para o atendimento
  4. Você recebe atualizações por e-mail
  5. O chamado é encerrado após a resolução
